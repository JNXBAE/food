# 🍽️ Jungle Foods Website

A modern, responsive, and user-friendly restaurant website built using **HTML5, CSS3, and JavaScript**. The website is designed for **Jungle Foods**, a multi-cuisine restaurant, providing customers with an attractive online experience to explore the menu, gallery, reviews, and contact information.

---

## ✨ Features

* 🏠 Responsive Home Page
* 🍛 Dynamic Menu with Categories
* 🖼️ Restaurant Gallery
* ⭐ Customer Reviews Section
* 📞 Contact Page
* 🔐 Admin Login Page
* 📊 Admin Dashboard
* 📱 Mobile-Friendly Design
* ⚡ Fast Loading and Clean UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* JSON (Menu Data)
* Local Storage
